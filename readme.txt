== Installation ==

1. Upload `target-heart-rate.zip` to the `/wp-content/plugins/` directory
2. Unzip archive
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Use this shortcode [target-heart-rates] in any page or post
